10. Condition

If costs increase, prices will rise.
If we act now, we can avoid delays.
If the system fails, operations will stop.
If demand continues, production will expand.
If employees agree, the policy will change.
If problems persist, we will reassess.
Inverted:
Prices will rise if costs increase.
We can avoid delays if we act now.
Operations will stop if the system fails.
Production will expand if demand continues.
The policy will change if employees agree.
We will reassess if problems persist.

15. Sequencing / Process

Model sentences:
First, we reviewed the data.
Next, the team discussed the results.
Then, the plan was revised.
After that, management approved the proposal.
During the meeting, several issues were raised.
Finally, the decision was approved.
Alternative forms:
After reviewing the data, we discussed the results.
Having completed the test, the team moved forward.
While analyzing the report, several errors were found.
Before signing the contract, all terms were reviewed.
Following the discussion, a decision was made.
Based on the feedback, the process was improved.
