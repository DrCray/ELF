LESSON 2

Explaining a Problem

Main Page Lesson 1 Lesson 2 Lesson 3 Lesson 4 Lesson 5 Lesson 6

Master the language for clearly describing challenges, issues, and obstacles in the workplace.

1. "We've run into an issue with..."
We've run into an issue with the payment processing system.
We've run into an issue with vendor delivery schedules.
We've run into an issue with the database connection.
We've run into an issue with securing the conference venue.
We've run into an issue with compatibility between the two platforms.
We've run into an issue with staff availability during the holiday period.
2. "The challenge we're facing is..."
The challenge we're facing is limited warehouse space for inventory.
The challenge we're facing is coordinating across multiple time zones.
The challenge we're facing is staying within the allocated budget.
The challenge we're facing is meeting the accelerated deadline.
The challenge we're facing is finding qualified candidates for the position.
The challenge we're facing is maintaining quality while scaling production.
3. "Unfortunately, there's been a delay due to..."
Unfortunately, there's been a delay due to customs clearance issues.
Unfortunately, there's been a delay due to unexpected technical difficulties.
Unfortunately, there's been a delay due to a key team member being out sick.
Unfortunately, there's been a delay due to waiting for regulatory approval.
Unfortunately, there's been a delay due to a supplier shortage.
Unfortunately, there's been a delay due to revisions requested by the client.
4. "What's complicating matters is..."
What's complicating matters is that we have conflicting requirements.
What's complicating matters is the ongoing restructuring in the department.
What's complicating matters is that two systems aren't communicating properly.
What's complicating matters is limited access to necessary data.
What's complicating matters is changing regulations in that market.
What's complicating matters is turnover in the project leadership.
5. "The root cause appears to be..."
The root cause appears to be outdated equipment in the facility.
The root cause appears to be insufficient training for new processes.
The root cause appears to be miscommunication between departments.
The root cause appears to be an error in the initial specifications.
The root cause appears to be unrealistic expectations set during planning.
The root cause appears to be a bug in the latest software update.
6. "This has created a situation where..."
This has created a situation where we can't meet the original timeline.
This has created a situation where costs are exceeding projections.
This has created a situation where customer complaints have increased.
This has created a situation where we need to reallocate team members.
This has created a situation where we're operating with reduced capacity.
This has created a situation where we need executive-level intervention.
