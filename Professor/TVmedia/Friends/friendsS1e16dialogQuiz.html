Basic Instructions
Overview:

  • Click any part of this form to edit it.

  • Click button to add a question/choices set

  • No typos in the answer, but extra spaces are okay.

  • Save to the browser's set download folder. Change file name if you like.

  • For practice tests (when the "exam" box is not checked) students can view their score as they practice.

  • As an exam (checked exam box), students can't see their score.

  • Students can save the exam as pdf and send to you (via email/group/social media).

  • When you recieve the tests, select/open or drag all files into the gradebook html file.
    The gradebook will automatically put the names and grades into the spreadsheet.

  • See details below

Details:
  • Click text to edit any part of this form.
    (for answer choices, double or triple click)
  • click add quSet to add another question/choices set. Edit the question set.
  • The answer: must match the correct-choice exactly for automatic scoring to work, although the number of spaces is not relevant.
    • For example "blue dog" " blue dog " "     b lu e   do g    " will all match correctly for scoring;
    • Misspelled or missing charactars are impossible to match,
      e.g. "blu dog" "blue dog" "bleu dog" will not match.

  • When you are finished, click Save/Download to save and download the new test as an html file. It will save to the browser's default download folder. You can change the file name to something more suitable.

    If is NOT selected (default), the form saves as an exercise, not an exam. Students can check their score as many times as they like.

    If is selected the form saves as an exam, students can not check their score.
    Some options are:

    • Send the file via email, group chat, or other social media to your students. They can open it in their web browser (thoroughly tested on Firefox, but does work on major broswers and mobiles).
    • Post it on a website. Give students the link and they can take the test online.
    • Put the html file on your cloud server or other file-sharing server, give students the link, and the students can download it and open it in their browser.

  • When your students finish the exam, they can click on print and save the file as PDF and send to you via email, group chat, social media or other file-sharing method.

    When saving as PDF students must use the button from the exam page Print/Save PDF , NOT the browswer's file menu, otherwise, scores won't be calculated.
